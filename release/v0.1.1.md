## discord-agent-hub v0.1.1

Operational polish release for teaching and shared-community use of `discord-agent-hub`.

### Highlights

- Usage reports now show resolvable Discord member display names alongside user IDs
- Interpreter example refined for thread speaker prefixes and practical Discord use
- Assistant transcript rendering no longer injects assistant speaker prefixes into model history
- Anthropic usage normalization now reports total tokens as `input + output`

### Notes

- The hub still assumes a single bot process on one host.
- Uploaded documents are text-extracted; embedded images inside documents are not extracted yet.
- `loglm` importer work remains on the `loglm` side.

### Platform Status

- Tested:
  - macOS
  - Ubuntu server operation with OpenAI, Anthropic, and Gemini APIs
- Untested:
  - Windows
  - multi-process deployment
