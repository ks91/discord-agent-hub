## discord-agent-hub v0.1.3

Learning- and analysis-oriented release for teaching, imitation, and readable session logs.

### Highlights

- Added `/agent-show-full` to display full public agent instructions
- Long full-instruction responses are now split across Discord interaction followups
- Structured logs now record a normalized top-level `model` on each assistant response
- Added Markdown rendering for exported JSONL event logs:
  - `scripts/render-events-md.py input.jsonl`
  - `scripts/render-events-md.py input.jsonl output.md`
- The event-log renderer now works directly from the repository root

### Notes

- `/agent-show` remains the short preview command
- `/agent-show-full` still respects `public_instructions: false`
- Event logs now make per-response model changes easier to inspect during research or teaching

### Platform Status

- Tested:
  - macOS
  - Ubuntu server operation with OpenAI, Anthropic, and Gemini APIs
- Untested:
  - Windows
  - multi-process deployment
